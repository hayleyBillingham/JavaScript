Prevent Default

  • The preventDefault() method cancels the event if it is cancelable
  • The default action that belongs to the event will not occur.
  • Not all evemts are cancelable. Use the cancelable property to find out if an event is cancelable
  • The prventDefault() methods does not prevent further propagation of an event through the DOM. Use the stopPropagation() method to handle this

Examples

  • Clicking on a submit button, prevent it from submitting a form
  • Clicking on a link, prevent the link from following the URL
