Equality Operators

  • Equality operators lets us compare if two values are equal
  • Inequality operators lets us know if two values are not equal
  • If a condition is eqaual then a code block will execute
  • if the condition is == then it is the value they look for being equal
  • if the condition is === then its the value and the type of variable is compared - this is strict equlaity operator
  • the strict eqaulity operator is best practice
  • != is not equal to - will do type conversion
  • !== is double not equal to this does not do any type conversion
  • normally these statements appear in if statements, while loops or other places where you expect conditionals
