Callbacks

  • A callback is a function passed as an argument to another function
  • This technique allows a function to call another function
  • a callback function can run after another function has finished

Function Sequence

  • JavaScript functions are executed in the sequence they are called not in the sequence they are defined

Sequence Control

  • Sometimes you would like to have better control over when to execute a function
  • Suppose you want to do a calculation, and then display the result. You could call a calculator function (myCalculator), save the result, and then call another function (myDisplayer) to display the result
  • Or you could call a calculator function (myCalculator), and let the calculator function call the display function (myDisplayer)
