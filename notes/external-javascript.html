External JavaScript

JavaScript can be placed in external files

Are practical when the same code is used in many different web pages

JavaScript files have the file extension .js.

To use an external script, put the name of the script in the src (source) attribute of a <script> tag:

You can place an external script reference in <head> or <body> The script will behave as if it is located exactly where the <script> tag is located.

External scripts cannot contain <script> tags

External scripts allow the JavaScript code to become re-usable

Example

            <script src="myScript1.js></script>

Advantages

Placing scripts in external files has some advantages.

  • It separates HTML and code
  • It makes HTML and JavaScript easier to read maintain
  • Cached JavaScript files can speed up page loads

External scripts can be referenced with a full URL or with a path relative to the current web page.

Example

                
                    <script> src="https://www.w3schools.com/js/myScript.js"></script>
