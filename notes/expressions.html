Expressions

  • Expressions evaluate to values
  • Expression can concatenate strings
  • You can have a variable included in an expression
  • Expressions can also evaluate to true or false

Example 1 - price

var total = price - (price & discount/100));

Example 2 - concatenate

var greeting = "Dear" + "Reader" + ","

Example 3 - varible

var = "super" + "cali" + youKnowTheRest;

Example 4 - Boolean expressions

age < 14

cost >= 3.99

animal == "bear"
