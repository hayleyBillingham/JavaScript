Statements

  • JavaScript supports different types of processing instruction types known as statements
  • Each statement specifies a small part of computation and together
  • all the statements create the behaviour of a page
  • There a many types of statements
    Assignment
    Where a value is assigned to a variable
    For Loops
    Processes a script over a number of iterations
    If-Else conditional statement
    This checks a condition to see whether the script block will be executed
    The Switch Statement
    Checks for a value in a given set and execute the script block associated with that value
  • Each type of statement has certain syntax requirements
    • A statement can be terminated by using the semicolon (;)
    • Using a semicolon in not required in JavaScript unless you type many statements on the same line
    • A termination of statements can be done by using a line break
    • It is good practice to use a semi colon for termination of statements
