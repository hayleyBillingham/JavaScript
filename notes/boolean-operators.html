Boolean Operators

  • A boolean operaots is used in a boolean expression
  • A boolean operator returns a true or false value
  • There are two kinds of boolean operators:
    • Comparison Operators
    • Logical operators

Comparison Operators

  • Comparison operators compare two values
  • < means less than
  • > means greater than
  • == means equal to
  • === means exactly equal to
  • <= means less than equal to
  • >= means greater than equal to
  • != means not equal to

Logical Operators

  • Logical operators combine two boolean expressions to create one boolean result (true or false).
  • || means OR, results in true if either of the two expressions is true
  • && means AND, results in true if both of the two expression is true
  • Another logical operator is NOT which acts on one boolean expression(rather than two)
  • ! means NOT, results in true if the expression is false
