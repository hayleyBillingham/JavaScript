Prototypes overview

  • A prototype is a object that exists on every function in javaScript
  • Objects do have a prototype but they do not have a prototype property
  • A function's prototype is the object instance that will become the prototype for all objects created using this function as a constructor
  • A Object's prototype is the object instance from which the object is inherited
  • A function and a object prototype is actually an instance of an object in memory
  • When a function has a created a new prototype, object is created in memmory and attached to the function behind the scenes
  • If that function is then used as a constructor function with the new key word, the new object that has been created has a proto property that is pointing to the same object in memory that is the functions prototype
