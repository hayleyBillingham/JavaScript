JavaScript Overview

  • JavaScript is the programming language of HTML and the Web
  • JavaScript is most well-known as the scripting language for web pages
  • JavaScript is a lightweight interpreted or just in time compiled programme language
  • JavaScript is supported in all modern browesers
  • JavaScript can also be used for non-browser enviroments such as Node.js, Apache CouchDB and Adobe Acrobat
  • JavaScript is prototype-based, multi-paradigm, dynamic language, supporting object-oriented, imperative and declarative programming styles.
  • The standard for JavaScript is ECMAScript
  • JavaScript is a cross-platform, object-oriented scripting language allowing you to add behaviour and used to make webpages interactive
  • There are more advanced server side versio ns of JavaScript such a Node.js which allows you to add more functionality to a website
  • JavaScript lets you create behaviour in your web pages
  • Javascript gives you a away to add prograaming to your page so you can compute, react, draw, communicate, alter, alertm, update, chance etc
  • You can place JavaScript in its own file to be included in your page
  • JavaScript is easy to learn.
  • JavaScript is one of the 3 languages all web developers must learn
    1. HTML to define the content of web pages
    2. CSS to specify the layout of web pages
    3. JavaScript to program the behaviour of web pages
  • In HTML, JavaScript code must be inserted between <script> and <script/>
  • A JavaScript function can be placed in the <head> section of an HTML page
  • ECMAScript is the official name for JavaScript
  • JavaScript is case sensitive
  • JavaScript is a high level interpreted programming language
  • Interpreted language means you dont have to compile the code before you execute it, it is interpreted at run time
  • A high level language means it is a high abrstaction from the computer, so it is easier to use
